\doxysubsection{Detailed Description}
Головний клас програми для роботи зі щасливими числами. 

Дана програма дозволяє користувачу обрати один із варіантів:\+
\begin{DoxyItemize}
\item знайти щасливе число, яке змінюється 4 рази,
\item знайти щасливе число, яке змінюється 7 разів,
\item одразу отримати фінальну відповідь,
\item завершити виконання програми.
\end{DoxyItemize}

Щасливими числами вважаються числа, що складаються лише з цифр 4 і 7, та мають задану кількість переходів між різними цифрами.
